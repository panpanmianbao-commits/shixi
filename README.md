时隙

一个有点可爱的计时器 App。写它是因为市面上的番茄钟都太像工具了，我想让倒计时这件事变得稍微浪漫一点。
目前还在慢慢完善中，功能基本能跑，但还有不少想加的东西。
它能干嘛

两种计时模式
普通倒计时：手动输时间，或者点一下 5分/10分/25分 的快捷按钮
番茄钟：专注 + 休息循环，支持自定义时长和轮数，也有几个预设模板（经典 25+5、深度 50+10 之类）

主题系统
倒计时不是干巴巴的数字在跳，而是有主题的。目前做了 8 个每个主题有自己的配色、进度条颜色，还有随时间推进会变化的文案。比如用「红叶の便り」的时候，你会看到「一叶知秋…」「红叶纷飞…」「秋思满怀」这样的状态描述，最后结束时还会掉落一片红叶，攒够 30 片可以「押し叶」（虽然这个功能我还没想好具体怎么呈现，先攒着）。

背景音乐
内置了几个在线电台，主要是古典、民谣、怀旧金曲之类的。可以切台、调音量、开关自动轮播。专注的时候有点背景音会舒服很多。
成就墙

完成一些特定条件会解锁成就，比如第一次计时、专注满 1 小时、用过 3 种不同主题、番茄钟完成 4 轮以上等等。没什么实际奖励，就是给自己一点正反馈。
用户系统

可以注册账号、登录、上传头像。数据存在本地 UserDefaults 里，没有后端，所以换设备就没了。以后可能会考虑加 iCloud 同步。

技术栈

SwiftUI
Combine（Timer）
AVFoundation（音频播放）
PhotosUI（头像上传）
运行

需要 Xcode 15+，iOS 17+。
bash

直接跑就行，没有第三方依赖。

碎碎念

这个 App 的名字「时隙」取自日语「時隙（じげき）」，意思是时间的缝隙。我希望它能帮你抓住那些容易被浪费掉的小段时间，哪怕只是 5 分钟，也能做点有意义的事。
如果你用了觉得哪里别扭，或者有什么想加的主题，欢迎提 Issue 或者 PR。
